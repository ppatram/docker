# Video 52 of 59 - Kubernetes Operators

📺 https://www.youtube.com/watch?v=mTC3UZ8bHJc

## Overview

This video covers Kubernetes Operators — what they are, how they differ from traditional deployment approaches, and their key components (CRD, CR, Controller). Operators automate Day 2 operations like backup, restore, updates, and scaling. The video demonstrates installing the Operator Lifecycle Manager (OLM), installing the cert-manager operator from OperatorHub, creating an Issuer, creating a Certificate, and verifying the operator manages certificate issuance and renewal automatically.

## Terminal Sessions

### Install OLM (Operator Lifecycle Manager)

```bash
curl -sL https://github.com/operator-framework/operator-lifecycle-manager/releases/download/v0.28.0/install.sh | bash -s v0.28.0
```

### Install cert-manager operator from OperatorHub via OLM

```bash
kubectl create -f https://operatorhub.io/install/my-cert-manager.yaml
```

### Wait for OLM pods to be ready

```bash
kubectl get pods -n olm
```

### Check ClusterServiceVersion (CSV) status

```bash
kubectl get csv -n operators
# Watch for status to change from pending -> installing -> succeeded
watch kubectl get csv -n operators
```

### Verify operator pods

```bash
kubectl get pods -n operators
```

### Verify cert-manager CRDs installed

```bash
kubectl get crd | grep cert-manager
```

### Create an Operator Group

```bash
kubectl apply -f og.yaml
```

### Check custom resources (before creating them)

```bash
kubectl get issuer
kubectl get certificate
```

### Create the self-signed Issuer

```bash
vi issuer.yaml
kubectl apply -f issuer.yaml
kubectl get issuer
```

### Create a Certificate

```bash
vi cert.yaml
kubectl apply -f cert.yaml
```

### Verify certificate and secrets

```bash
kubectl get issuer
kubectl get certificate
kubectl get secret
kubectl describe secret my-app-tls
```

### Check operator logs for certificate status

```bash
kubectl logs -n operators -l app=cert-manager --tail=100
```

### Cleanup

```bash
kubectl delete -f cert.yaml
kubectl delete -f issuer.yaml
kubectl delete -f og.yaml
```

## YAML Files

### Operator Group (og.yaml)

```yaml
apiVersion: operators.coreos.com/v1
kind: OperatorGroup
metadata:
  name: my-operator-group
  namespace: operators
spec:
  targetNamespaces:
    - operators
```

### Self-Signed Issuer (issuer.yaml)

```yaml
apiVersion: cert-manager.io/v1
kind: Issuer
metadata:
  name: self-signed-issuer
spec:
  selfSigned: {}
```

### Certificate (cert.yaml)

```yaml
apiVersion: cert-manager.io/v1
kind: Certificate
metadata:
  name: my-app-certificate
spec:
  secretName: my-app-tls
  dnsNames:
    - myapp.example.com
  issuerRef:
    name: self-signed-issuer
    kind: Issuer
```
