## Overview

This video discusses the CKA exam curriculum changes that became effective on February 18, 2025. The instructor outlines what topics were added, changed, removed, and the new exam weight distribution. The exam now focuses 30% on troubleshooting and 25% on architecture (55% combined). The exam cost increased by 10%.

> **Note:** This video is purely conceptual/informational with no terminal commands or hands-on demos.

## Key Changes Summary

### Increased Focus Topics (already covered in series)
- Autoscaling
- Affinity groups / Node Affinity
- Pod Security
- Taints and Tolerations
- DNS Troubleshooting
- Dynamic Volume Provisioning
- Storage Classes
- Volume Types, Access Modes, Reclaim Policy

### Newly Added Topics
1. **Helm** (package manager for Kubernetes)
2. **Kustomize** (manifest management)
3. **CRDs and Operators** (Custom Resource Definitions, Custom Resources)
4. **CNI, CSI, CRI** (Container extension interfaces)
5. **Gateway API** (newer version of Ingress)
6. **Pod Admission**
7. **High Availability Control Plane**
8. **StatefulSets**
9. **Priority Classes**

### Removed Topics
- etcd backup and upgrade (removed from exam, but still useful to know)
