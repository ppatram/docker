# CKA 2024 Course Introduction and Announcements

📺 [Watch on YouTube](https://www.youtube.com/watch?v=6_gMoe7Ik8k)

## Overview

This is the introductory video (Day 0) for the CKA 2024 series by Piyush. It covers four announcements: course structure (40 videos + bonus), Discord community for assistance, weekly live Q&A sessions, and the "40 Days of Kubernetes" challenge. The series covers the full CKA curriculum with hands-on demos, diagrams, and code snippets.

## Key Points

- Series has 40 videos total + a bonus exam tips video
- Video 40 is a real-time project covering probes, deployments, services, ingress, etc.
- Resources available in GitHub repo: `resources.md` and `task.md` per video
- 3-4 videos published per week
- Discord community server for help (channel: help-cka2024)
- Weekly Sunday Q&A sessions
- "40 Days of Kubernetes" challenge with prizes
