  what is the root password for the default qemu image

  There is no default root password — most cloud/qemu images (Ubuntu, CentOS, Debian) disable password login entirely and use SSH keys only.

  To reset it without knowing it:

  virt-customize -a /path/to/disk.qcow2 --root-password password:yourpassword

  Or check how the VM was created — if you used virt-install with --cloud-init, the credentials were set there.
